## Requerimentos

    Node 9.5
    MongoDB ^3.4.9
    Nodemon (para testes)


## Para instalar

    git clone git@github.com:genezysg/dsfbit.git
    cd dsfbit
    npm install

## Scripts:

    npm run dev - Executa em modo de desenvolvimento (com nodemon)
    npm run test - Executa os testes
    npm run cover - Cria os arquivos de cobertura de teste
